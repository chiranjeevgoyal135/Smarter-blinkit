## 🎨 Design System Updates

### **Colors**
- **Primary:** `#f6a623` → `#f97316` (Orange gradient)
- **Secondary:** `#3b82f6` → `#2563eb` (Blue gradient)
- **Success:** `#10b981` (Emerald)
- **Purple:** `#7c3aed` → `#a855f7` (Purple gradient)
- **Background:** Dark gradients with subtle patterns

### **Typography**
- **Headings:** Increased font weights (800-900)
- **Body:** Better line heights and spacing
- **Labels:** Uppercase with letter spacing for emphasis

### **Spacing**
- **Cards:** Increased padding (24-28px)
- **Gaps:** Larger gaps between elements (20-24px)
- **Margins:** More breathing room

### **Effects**
- **Glassmorphism:** `backdrop-filter: blur(20px)`
- **Shadows:** Larger, softer shadows
- **Borders:** Subtle colored borders with transparency
- **Animations:** Smooth cubic-bezier transitions
